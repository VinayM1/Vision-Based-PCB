# Vision-Based Embedded PCB

This project is a custom-designed PCB integrating a microcontroller, camera interface, power management, and communication modules into a compact layout.

## Overview
The goal of this project was to design a practical and compact embedded system capable of processing visual input and interacting with external systems.

## Features
- Camera interface integration
- Custom PCB design (2-layer)
- Power management (battery + regulation)
- USB communication
- Compact layout optimized for real-world use

## Challenges
- Dense component placement
- Routing high-speed signals
- Power integrity and decoupling
- Space optimization

## Applications
- Wearable electronics
- Vision-based systems
- Embedded sensing devices

## Status
Design completed, currently working on further miniaturization and optimization.

## Tools Used
- EasyEDA
- ESP32-S3R8
